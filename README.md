# ShiftPilot Prototype v8

This version replaces the approximate headcount recommendation with a target-tested recommendation.

ShiftPilot now generates and scores candidate rotas, increasing headcount until the selected weekly target is actually achieved under the current opening hours, shift limits, live-headcount floor, part-time setup and optimisation rules.

The displayed recommendation is therefore the lowest tested headcount that meets the chosen target, rather than a simple hours-based estimate.
